# Boilerplate code for a project using Symfony as backend and React with Vite for frontend

## Requirement
- PHP
- Scoop
- Composer
- Symfony CLI
- Node.js

## 1. Install dependencies:
  ``` composer

  composer install
    
  ```
   
  ```npm

  npm install
   
  ```
## 2. launch the project

 ```composer
 
 symfony serv:start -d
 
 ```
 
 ```npm
 
 npm run dev
 
 ```
